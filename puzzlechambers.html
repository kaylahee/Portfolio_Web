Puzzle Chambers Game

3D 방탈출 게임

프로젝트 개요

이 프로젝트는 퍼즐 방탈출 게임으로, 플레이어는 총 3개의 방에서 주어진 문제를 풀어 탈출해야 합니다. 각 방에는 도전 과제가 있으며, 일부 방은 서로 연계된 요소를 포함하고 있습니다.

  • 1번 방: 퍼즐 조각을 보드에 맞추어 문제를 해결합니다.
  • 2번 방: 두 개의 무게 저울을 이용하여 정확한 무게를 맞추어야 합니다.
  • 3번 방 : 랜턴을 사용하여 빛을 통해 숨겨진 단서를 찾아 탈출합니다.
  • 4번 방 : 3번 방에서 얻은 힌트로 알맞는 물체를 골라 탈출합니다.

주요 구현 내용

1번 방 구현

  • 플레이어가 E 버튼을 눌러 물체를 집을 수 있도록 구현했습니다.
  • 퍼즐 조각을 보드에 갖다 대면 box collider가 겹칠 때 스냅되도록 하여 자연스럽게 배치할 수 있도록 했습니다.
  • 퍼즐에 color와 shape 속성을 지정하여 올바른 위치에 스냅되었는지 비교할 수 있게 구현했습니다.

2번 방 구현

  • 무게 측정 장치에 올려진 큐브들의 Mass 값을 실시간으로 합산하여 총 무게를 계산하고, 두 장치의 총 무게가 동일할 경우 문이 열리도록 구현했습니다.
  • PuzzleManager 스크립트 내에서 Mass 변수를 활용해 각 큐브별로 무게를 개별 설정할 수 있도록 구현했습니다.

3번 방 구현

  • 랜턴을 장착한 후 R 키를 누르면 불빛이 켜지고, 불빛에 해당하는 콜라이더를 활성화하여 벽에 숨겨진 알파벳이 드러나도록 구현했습니다. 불빛이 꺼지면 콜라이더도 비활성화되어 알파벳이 다시 보이지 않게 됩니다.

4번 방 구현

  • 3번 방에서 얻은 힌트로 알맞는 물체를 골라 탈출할 수 있도록 구현했습니다.
